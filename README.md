## 🌌 NebulaShare
## P2P File Sharing System

NebulaShare is a decentralized peer-to-peer (P2P) file sharing system that enables fast, secure, and direct file transfers between devices without relying on a central server.

## 🚀 Features
🔗 Decentralized Architecture – No central server required  
⚡ High-Speed Transfers – Parallel chunk-based file sharing  
🔐 Secure Sharing – Encrypted data transmission  
🌐 Peer Discovery – Automatic detection of available devices  
📦 Reliable Transfers – Integrity checks ensure data accuracy  
💻 Cross-Platform Support – Works across multiple devices  
🧠 How It Works  
Devices connect through a P2P network  
Peers are discovered automatically  
Files are split into smaller chunks  
Chunks are transferred in parallel  
Receiver reassembles the file securely
